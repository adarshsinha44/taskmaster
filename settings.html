Principles shown here

  • Universal design: settings support different readability needs.
  • User control: preferences can be changed without affecting the main flow.
  • Consistency: controls use the same styling and spacing as every other screen.

Feedback applied

  • Simple layout options were added after peer feedback about visual comfort.
  • Language support was fixed to demonstrate inclusive design more clearly.
  • All accessibility controls were grouped in one screen to avoid confusion.
